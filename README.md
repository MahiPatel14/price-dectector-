## 🛠️ Installation

Clone this beast:

```bash
git clone <your-repo-link>
cd PYTHONCIAP
```

Create virtual environment:

```bash
python -m venv venv
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```
